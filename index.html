About

Fresnel is an exploit-free burn+reflect+liquidity token on Binance Smart Chain (BSC).

  • Each transaction has a 3% fee:
  • 1% is burned immediately.
  • 1% is reflected proportionally to all holders.
  • 1% is automatically locked into a PancakeSwap liquidity pool.

Fresnel is not a safemoon clone. The code is written from scratch, based on liquid.defi and reflect.finance, omitting potential exploits. The contract is crystal clear, is about 120 lines, and is intended to be read by clients.

There is only limited owner-only functionality for compatibility with DxSale. Deployer is in the same position as an adopter once all tokens are sent to DxSale for ICO and ownership is renounced.
